目录详解

  • build：目录是一些webpack的文件，配置参数什么的，一般不用动
  • src：项目开发相关源码
  • static：生成好的文件会放在这个目录下。
  • test：测试文件夹，测试都写在这里
  • .babelrc： babel编译参数，vue开发需要babel编译
  • .editorconfig：不知名开发工具配置文件
  • .eslintrc.js： eslint配置文件，用以规范团队开发编码规范，大中型项目很有用
  • .gitignore：git过滤配置
  • index.html：主页
  • package.json： 项目文件，记载着一些命令和依赖还有简要的项目描述信息
